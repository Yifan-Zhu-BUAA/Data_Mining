# 图像聚类任务 - 快速开始指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 运行程序

```bash
python main.py
```

## 3. 查看结果

运行完成后，结果将保存在 `results/` 目录下：

- `features.npy`: 提取的特征矩阵
- `comparison_results.csv`: 各聚类方法的比较结果
- `detailed_report.txt`: 详细的文本报告
- `visualization_*.png`: 聚类结果可视化图像

## 4. 生成LaTeX报告

使用XeLaTeX编译报告：

```bash
xelatex report.tex
xelatex report.tex  # 需要编译两次以生成目录
```

或者使用在线LaTeX编辑器（如Overleaf）编译。

## 5. 修改参数

可以在 `main.py` 中修改以下参数：

```python
# 特征提取方法
feature_method = 'resnet'  # 可选: 'resnet', 'hog', 'histogram', 'combined'

# 聚类数量
n_clusters = 6

# 设备选择
device = 'cuda'  # 或 'cpu'
```

## 注意事项

1. 确保数据集路径正确：`../cluster/Cluster/dataset/`
2. 确保标签文件存在：`../cluster/Cluster/cluster_labels.json`
3. 如果有GPU，程序会自动使用GPU加速
4. 完整流程约需10-30分钟（取决于硬件配置）
